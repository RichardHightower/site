Boon

 
 

Simple opinionated Java for the novice to expert level Java Programmer. Low Ceremony. High Productivity. A real boon to Java to developers!


Boon came from my frustration when moving back and forth between Groovy, Ruby, Python and Java. I do a lot of contract programming and consulting, and I do not always get to pick my programming language. I found that I keep dragging code tricks around from project to project to fill the gaps for simple things like slice notation or reading a file quickly or doing an HTTP post.

 

Low Ceremony. High Productivity.
